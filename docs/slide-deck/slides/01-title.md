# Slide 01: 表紙

## Role

最初に「引き継ぎって本当に届いているのか？」と問いかけ、聞き手の関心を引く。

## Audience reaction

自分たちの引き継ぎにも起きている問題かもしれない、と感じてもらう。

## Display text

```text
引き継ぎ、
ちゃんと届いていますか？

Handover Notifier
```

## Visual direction

- 通知を受け取る大学生風メンバー
- 実アプリアイコンまたはアプリ名
- Slack/Gmail風の通知カード
- 明るく期待感のある背景

## Required assets

- 背景・人物はGPT Imagesで素材生成
- アプリアイコンがある場合は実素材を使用
- Slack/Gmail風の通知カードはHTML/CSSまたは実アプリに矛盾しないUI要素

## Avoid

- 画像内に日本語テキストを入れない
- 架空の機能説明を足さない
- 表紙から企業向けSaaSのように硬く見せすぎない

## Implementation notes

- 文字はHTML/CSSで配置する
- 全体のビジュアルトーンを確認する試作対象にする
- アプリ名 `Handover Notifier` は読みやすく配置する
