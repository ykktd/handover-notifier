# Slide 09: まとめ

## Role

最初の問題提起に戻り、導入価値を一言で締める。

## Audience reaction

Handover Notifier の価値を「探す引き継ぎから、届く引き継ぎへ。」という言葉で覚えてもらう。

## Display text

```text
探す引き継ぎから、
届く引き継ぎへ。

必要な情報を、必要な日に。
```

## Visual direction

Before / After 構成。

```text
Before：資料を探す
After：通知が届く
```

## Required assets

- 背景と人物はGPT Images
- Before/Afterラベルと本文はHTML/CSS
- 最後にアプリ名またはアイコンを配置

## Avoid

- 新しい機能説明を追加しない
- 「見逃しゼロ」などの断定表現を入れない
- 画像生成に日本語テキストを含めない

## Implementation notes

- 最初の問題提起と呼応する構成にする
- コアメッセージを最も目立たせる
- Before/Afterラベルと本文はHTML/CSSで固定する
