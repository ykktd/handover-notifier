# Slide 07: 導入メリット

## Role

機能説明から、聞き手にとってのメリットへ戻す。

## Audience reaction

ずっと資料を確認し続ける負担が減り、必要な日に思い出せる安心感を持ってもらう。

## Display text

```text
もう、ずっと
追い続けなくていい。

・抜け漏れを防ぎやすい
・確認の心理的負担が減る
・引き継ぎ品質が安定する
```

## Visual direction

- 通知を見て安心している後輩メンバー
- チェック済みタスク
- 整理されたカレンダーやタスクカード

## Required assets

- 人物・背景はGPT Images
- 通知カードと文字はHTML/CSS

## Avoid

- 「見逃しゼロ」と断定しない
- 企業向けSaaSのような過度な業務改善表現にしない
- 効果を保証するような表現にしない

## Implementation notes

- メリットは聞き手の心理的負担軽減として見せる
- 通知カードは補助的に使い、テキストの読みやすさを優先する
