# Slide 02: 従来の課題

## Role

「資料があるだけでは、必要なタイミングで見てもらえない」という問題を提示する。

## Audience reaction

資料は作ったのに見に行かない、というサークル引き継ぎのありがちな状況に共感してもらう。

## Display text

```text
資料はある。
でも見に行かない。

・年度初めに資料を作成
・Google Drive に置いて共有
・必要な時は自分で探す
```

## Visual direction

- Drive風のクラウドストレージに資料が埋もれている
- どの資料を見るべきか迷っている後輩
- 散らかったファイル、フォルダ、検索アイコン

## Required assets

- 背景と人物はGPT Images
- 文字はHTML/CSS

## Avoid

- 背景内に文字を入れない
- Google Driveの正確なロゴやUIを生成画像で作らない
- 問題を個人の怠慢として責める表現にしない

## Implementation notes

- 箇条書きは3点に留める
- 迷いやすさはビジュアルで補い、テキストは短く保つ
